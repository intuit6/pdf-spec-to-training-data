📄 PDF Spec to Training Data

将 PDF 规范文档快速转换为机器学习训练数据集

🚀 快速开始

git clone https://github.com/yourusername/pdf-spec-to-training-data.git
cd pdf-spec-to-training-data
pip install -e ".[all]"
python scripts/generate_dataset.py --config config/config.yaml

详细教程请查看 项目 README 和 QUICKSTART.md

✨ 核心特性

  • 📝 PDF 文本提取 - 高精度提取文字型 PDF 内容
  • 📊 表格识别 - 自动提取有线表和无线表
  • 🔍 智能分块 - 按章节、段落或固定长度分割
  • 📤 多格式输出 - JSONL、CSV、TXT、Parquet
  • ⚙️ 配置驱动 - YAML 配置文件控制
  • 🚀 批量并行 - 支持多进程加速

📦 支持的功能

PDF 类型

  • ✅ 文字型 PDF
  • ✅ 扫描型 PDF（需 OCR）
  • ✅ 混合型 PDF

表格支持

  • ✅ 有线表（Lattice）
  • ✅ 无线表（Stream）
  • 🔄 自动检测策略

输出格式

JSONL
CSV
TXT
Parquet
JSON

🔧 使用场景

  • 🏗️ LLM 训练数据准备（ChatGLM、Llama、GPT 等）
  • 📚 RAG 知识库构建和向量化
  • 📊 技术文档结构化和分析
  • 🔄 版本规范和差异对比
  • 🏷️ 数据标注和术语提取
